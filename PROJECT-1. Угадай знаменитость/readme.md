# Проект:  Угадай знаменитость

Данный проект посвящен разработке нейросети для классификации изображений пяти различных личностей. Работа выполнена с использованием глубокого обучения и метода трансферного обучения (Transfer Learning).

## Описание проекта
Цель работы — создать модель, способную с высокой точностью (Accuracy > 0.85) определять, кто изображен на фотографии из заданного набора данных.


## Основные этапы работы:
1. **Загрузка данных:** Использование `DataLoader` для эффективной подачи данных в модель.
2. **Предобработка:** Применение аугментаций (`RandomResizedCrop`, `RandomHorizontalFlip`, `ColorJitter`) для повышения устойчивости модели к изменениям на входных изображениях.
3. **Transfer Learning:** Использование архитектуры `ResNet18`, предобученной на наборе данных ImageNet.
4. **Модификация:** Адаптация выходного слоя модели под 5 целевых классов.
5. **Обучение:** Использование оптимизатора `Adam` и функции потерь `CrossEntropyLoss`.
6. **Визуализация:** Отрисовка примеров из датасета и итоговых предсказаний модели.

---

### Описание результатов работы модели:
В данном проекте была решена задача многоклассовой классификации (5 классов/личностей) с использованием принципов трансферного обучения (Transfer Learning).

*   **Модель:** В качестве базовой архитектуры была выбрана предобученная сверточная нейросеть ResNet18 (веса ImageNet).
*   **Адаптация:** Финальный полносвязный слой (`model.fc`) был заменен на новый, имеющий 5 выходных нейронов (по количеству наших классов). Базовые слои сети были заморожены, обучался только новый классификатор.
*   **Аугментация:** Для борьбы с переобучением и лучшей обобщающей способности применены аугментации данных: случайные обрезки (`RandomResizedCrop`), отзеркаливания (`RandomHorizontalFlip`) и изменение цвета (`ColorJitter`).
*   **Результаты:** Обучение происходило с помощью алгоритма Adam и функции потерь CrossEntropy. Модель успешно сошлась, достигнув значения Accuracy на валидационной выборке: 0.9956, что уверенно превышает целевой порог в 0.85.
*   **Визуализация:** Предсказания модели подтверждают, что сеть уверенно распознает признаки лиц из обучающего набора и корректно классифицирует изображения из валидационного датасета. Ошибки (если они есть на визуализации, отмечены красным) связаны с нестандартными ракурсами и недостаточной освещенностью некоторых исходных фотографий.
