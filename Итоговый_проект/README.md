# Итоговый проект: Определение уязвимых групп населения

## Основные цели и задачи проекта
Цель проекта — на основе данных Росстата и ЕМИСС выделить группы (кластеры) регионов России с похожими социально-экономическими профилями, чтобы определить наиболее уязвимые субъекты, требующие приоритетной адресной социальной поддержки. 

Основные задачи:
1. Сбор и очистка разрозненных данных из различных источников.
2. Проведение исследовательского анализа данных (EDA) и выявление корреляций.
3. Построение ML-модели кластеризации для сегментации регионов.
4. Статистическое подтверждение значимости полученных результатов.

## Краткая информация о данных
В работе используются официальные открытые данные:
* Уровень преступности среди несовершеннолетних.
* Демографические показатели (доля детей и пенсионеров в малоимущих семьях).
* Экономические показатели (ВРП на душу населения, уровень бедности).
Данные прошли стадию глубокой предобработки: очистка от технических строк (федеральные округа, сельсоветы), приведение названий регионов к единому стандарту и объединение в единый датасет (Merge).

## Этапы работы над проектом
1. **Подготовка среды и сбор данных:** Загрузка датасетов в формате `.xlsx` и `.csv`, отключение системных предупреждений.
2. **Предобработка данных:** Написание кастомной функции для стандартизации названий регионов, удаление пропусков, влияющих на качество модели.
3. **Исследовательский анализ (EDA):** Построение корреляционной матрицы для выявления линейных зависимостей (например, между бедностью и преступностью).
4. **Машинное обучение (Кластеризация):** * Нормализация признаков с помощью `StandardScaler`.
   * Разделение регионов на 3 кластера с использованием алгоритма `K-Means`.
   * Проверка стабильности решения путем сравнения с `AgglomerativeClustering`.
5. **Статистическая проверка:** Использование теста ANOVA для подтверждения статистической значимости различий между сформированными кластерами.
6. **Формирование выводов:** Выделение «кластера риска» и разработка рекомендаций по социальной поддержке.
