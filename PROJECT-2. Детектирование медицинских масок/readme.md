# 😷 Face Mask Detection (Детектирование медицинских масок)

Учебный проект по распознаванию медицинских масок на лицах людей с использованием современных алгоритмов компьютерного зрения.

## Описание проекта
Представьте, что вашей компании выпала возможность помочь миру путем автоматизации контроля ношения масок людьми. 
Цель данного проекта — разработать систему детектирования, а также сравнить две популярные архитектуры:
* **Двухуровневый детектор:** Faster R-CNN (на базе ResNet50)
* **Одноуровневый детектор:** YOLOv8

**Главная задача:** Достижение метрики `mAP > 0.85` (Mean Average Precision) на валидационной выборке.

## Датасет
Используется набор данных [Face Mask Detection с Kaggle](https://www.kaggle.com/andrewmvd/face-mask-detection).
Включает 853 изображения с аннотациями в формате PASCAL VOC (XML).
Размеченные классы:
- `with_mask` (в маске)
- `without_mask` (без маски)
- `mask_weared_incorrect` (маска надета неправильно)

*Особенности обработки:* В рамках проекта реализован парсер аннотаций из XML в структурированный DataFrame (CSV) с автоматическим исправлением некорректных bounding boxes, выходящих за границы изображений.

## 🛠 Стек технологий
* **Язык:** Python
* **Фреймворки глубокого обучения:** PyTorch, Torchvision, Ultralytics (YOLOv8)
* **Расчет метрик:** Torchmetrics (IoU, mAP)
* **Аугментация данных:** Albumentations
* **Работа с данными и визуализация:** Pandas, Matplotlib, OpenCV, Pillow

## Основные этапы выполнения
1. **Подготовка данных:** Чтение XML-аннотаций, перевод в CSV, анализ и базовая очистка разметки.
2. **DataLoader:** Создание кастомного загрузчика данных с применением аугментаций для PyTorch.
3. **Обоснование и визуализация:** Выбор предобученных на COCO моделей и отрисовка тестового батча данных с наложенными рамками.
4. **Обучение моделей:** Тонкая настройка (fine-tuning) Faster R-CNN и YOLOv8 на тренировочной выборке.
5. **Оценка качества:** 
   - Подсчет метрик Intersection over Union (IoU) и Average Precision (AP).
   - Построение PR-кривых (Precision-Recall).
   - Визуализация итоговых предсказаний на тестовых данных.

## Результаты и выводы
В ходе работы обе архитектуры (Faster R-CNN и YOLOv8) успешно преодолели целевой порог качества в `mAP > 0.85`. 
- **Faster R-CNN** продемонстрировала высокую точность локализации, в том числе на сложных перекрытиях.
- **YOLOv8** показала отличный баланс между высокой точностью и скоростью инференса (FPS), что делает её более предпочтительной для интеграции в системы контроля и видеонаблюдения в реальном времени.
